# 🏢 Pakistan Real Estate Market Analysis (EDA)

## 📌 Project Overview

This project is an Exploratory Data Analysis (EDA) of Pakistan’s real estate market using a dataset of 190,000+ property listings from Zameen.com. The goal is to extract meaningful insights that can help investors, agencies, and buyers make data-driven decisions.

---

## 🎯 Objectives

* Clean and preprocess real-world messy data
* Perform feature engineering (area conversion, time features)
* Detect and handle outliers
* Conduct univariate, bivariate, and multivariate analysis
* Answer real business questions using data

---

## 🗂️ Dataset Information

* Source: Zameen.com
* Records: 190,000+
* Format: CSV

### Key Features:

* Property Type
* Price (PKR)
* City & Location
* Area (Marla/Kanal)
* Bedrooms & Bathrooms
* Purpose (Sale/Rent)
* Date Added
* Agency & Agent

## 📊 Key Insights

* Karachi & Lahore show massive growth in high-value properties (>50M PKR)
* Karachi is dominated by flats (vertical living trend)
* Lahore & Islamabad prefer houses (independent living)
* Property prices increase with bedrooms/bathrooms but plateau after a threshold
* Weak correlation between property size and price → location is more important
* Peak listing activity occurs in June–July
* Islamabad has more rental listings than sales (unique market behavior)

---

## 💡 Business Takeaways

* Location plays a bigger role than size in property pricing
* Investors should target June–July for higher market activity
* Islamabad is ideal for rental-based investment strategies
* Karachi shows strong potential in apartment/flat investments
